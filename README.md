iMovie——爱电影，爱生活
=========================================

### 个性化电影推荐

- 数据来源：通过java的HTTPClient爬取豆瓣电影，Jsoup进行解析
- 数据库：采用开源数据库PostgreSQL
- 后台：采用Nodejs的Express框架
- 前端：html采用jade模版，css采用sass模版，javascript采用jquery库
- 版本管理：采用Git进行版本的更新与修改
- 服务器：Nginx

### 运行

```bash 
$ npm install
$ gulp watch
```
